OpenRCT2 is an open-source re-implementation of RollerCoaster Tycoon 2 that provides everything from the original game plus numerous improvements and additional features. The engine is written in modern C++ and supports contemporary platforms while maintaining gameplay compatibility with RCT2.

## Core Improvements

<CardGroup cols={2}>
  <Card title="Modern Platform Support" icon="desktop">
    Native support for Windows, macOS, Linux, and even Android/iOS platforms with better performance and stability than the original.
  </Card>
  
  <Card title="Multiplayer" icon="users">
    Cooperative multiplayer support allowing multiple players to build and manage parks together in real-time.
  </Card>
  
  <Card title="Increased Limits" icon="chart-line">
    Raised limits on entities, rides, and map size:
    - Up to 3200 misc entities (vs 1600 in original)
    - Larger map sizes supported
    - More guests (slow down threshold raised to 52000 from 7000)
  </Card>
  
  <Card title="Save File Compatibility" icon="floppy-disk">
    Loads original RCT2 and RCT1 save files, scenarios, and track designs. Supports Zstd compression for faster saving and smaller files.
  </Card>
</CardGroup>

## Enhanced Editing & Building Tools

<CardGroup cols={2}>
  <Card title="Advanced Construction" icon="hammer">
    - Build in pause mode
    - Place self-intersecting track designs
    - Footpath area dragging tool
    - Wall line dragging tool
    - Maze construction auto-removes trees
  </Card>
  
  <Card title="Clearance & Support Options" icon="wrench">
    - Disable clearance checks
    - Disable support limits
    - Place tracks at invalid heights
    - Show all operating modes
    - Show vehicles from other track types
  </Card>
  
  <Card title="Tile Inspector" icon="magnifying-glass">
    Advanced tile editing capabilities for modifying map elements, changing properties, and fine-tuning park layouts.
  </Card>
  
  <Card title="Zero Clearances" icon="crop">
    Ability to build scenery and rides with overlapping elements for creative and compact park designs.
  </Card>
</CardGroup>

## Gameplay Enhancements

### Guest & Staff AI Improvements

- Improved pathfinding and decision-making
- Staff use appropriate animations when waiting at level crossings
- Security guards walk slowly only in crowded areas
- Guests have proper happiness penalties for low energy, hunger, thirst, and toilet needs
- Better crowd management and performance

### Ride & Track Features

<CardGroup cols={2}>
  <Card title="Track Designs" icon="draw-polygon">
    - Rides with modern elements (zero G rolls, diagonal brakes) can be saved
    - Track designs can be shifted underground
    - Better preview rendering with correct path connections
  </Card>
  
  <Card title="Operating Options" icon="gear">
    - Unlock operating limits (fast lift hills)
    - Chain lifts on all track pieces
    - Disable train length limits
    - Arbitrary ride type changes
    - Powered launch mode adjustments
  </Card>
</CardGroup>

### RCT1 Feature Re-introduction

OpenRCT2 brings back several features from the original RollerCoaster Tycoon that were absent in RCT2:

- Mountain tool available in-game
- "Have Fun" objective
- Launched coasters without passing through the station
- Several toolbar buttons from RCT1

## Performance & Technical

### Optimization

- Multi-threaded rendering and game logic
- Faster sprite building and processing
- Improved map selection grid rendering
- Removal of 2000 animated tile limit
- More efficient pathfinding algorithms
- Better memory management

- Import RCT1 scenarios and data
- Load RCT Classic scenarios with packed objects
- Scenario previews in selection window

- Improved financial tracking and statistics
