## What is OpenRCT2?

OpenRCT2 is an open-source re-implementation of **RollerCoaster Tycoon 2** (RCT2), a construction and management simulation video game that simulates amusement park management. The gameplay revolves around building and maintaining an amusement park containing attractions, shops and facilities.

<img src="https://i.imgur.com/e7CK5Sc.png" alt="OpenRCT2 gameplay screenshot" />

### The Goal

As a park manager, you must try to make a profit and maintain a good park reputation whilst keeping your guests happy. OpenRCT2 allows for both scenario and sandbox play:

## History and Background

RollerCoaster Tycoon 2 was originally written by Chris Sawyer in x86 assembly and is the sequel to RollerCoaster Tycoon. The engine was based on Transport Tycoon, an older game which also has an equivalent open-source project, [OpenTTD](https://openttd.org).

OpenRCT2 began as a project to decompile and improve the original RCT2 game, bringing it to modern platforms and adding features that the community had wanted for years.

## Key Features

OpenRCT2 attempts to provide everything from RCT2 as well as many improvements and additional features:

### Classic Features Restored

OpenRCT2 re-introduces mechanics from the original RollerCoaster Tycoon that were not present in RCT2:

- Mountain tool available in-game
- The *"have fun"* objective
- Launched coasters (not passing through the station)
- Several additional toolbar buttons

<Note>
  OpenRCT2 requires the original RollerCoaster Tycoon 2 game files to play. You can purchase RCT2 from [Steam](https://store.steampowered.com/app/285330/RollerCoaster_Tycoon_2_Triple_Thrill_Pack/) or [GOG.com](https://www.gog.com/game/rollercoaster_tycoon_2).
</Note>

## Why Choose OpenRCT2?

### Active Development

OpenRCT2 is actively maintained by a passionate community of developers who continuously add new features, fix bugs, and improve performance.

### Open Source

As an open-source project licensed under the GNU General Public License version 3.0, OpenRCT2 is free to use and modify. The source code is available on [GitHub](https://github.com/OpenRCT2/OpenRCT2).

### Community Driven

- Active [Discord community](https://discord.gg/fsEwSWs) for support and discussion
- Regular updates from both stable and development branches
- Plugin system for custom modifications
- Translation support for multiple languages

### Backwards Compatibility

If you have the original RollerCoaster Tycoon and its expansion packs, you can point OpenRCT2 to these files to play the original RCT1 scenarios.

## System Requirements

<Warning>
  OpenRCT2 will only work on little-endian architectures. Most modern desktop and mobile devices use little-endian architecture.
</Warning>

### Minimum Requirements
- Operating System: Windows 7, Linux, macOS 10.13, Android, or BSD
- Memory: 2 GB RAM recommended
- Storage: 500 MB available space
- Graphics: OpenGL or DirectX support for hardware display mode

### Required
- RollerCoaster Tycoon 2 game files (from Steam, GOG, or original disc)
