## Reporting Bugs

<Steps>
  <Step title="Check Existing Issues">
    Before reporting a bug, search the [issue tracker](https://github.com/OpenRCT2/OpenRCT2/issues) to see if it's already been reported.
  </Step>
  
  <Step title="Gather Information">
    Collect relevant details:
    - OpenRCT2 version (stable or develop build number)
    - Operating system and version
    - Steps to reproduce the issue
    - Screenshots or save files if applicable
  </Step>
  
  <Step title="Submit Issue">
    Create a new issue on [GitHub](https://github.com/OpenRCT2/OpenRCT2/issues/new) with all the gathered information.
  </Step>
</Steps>

<Tip>
If you're able to fix the bug yourself, consider submitting a pull request! See our [Contributing Guide](/development/contributing) for more information.
</Tip>
